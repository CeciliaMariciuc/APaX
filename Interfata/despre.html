Cerinta

APaX (Auto Park Web Explorer) Recurgand la un API REST/GraphQL propriu, sa se realizeze un instrument Web de vizualizare adecvata si de comparare multi-criteriala a datelor publice privind parcul auto din Romania pe ultimii 5 ani. Statisticile, plus vizualizarile generate – minim 3 maniere, plus cele cartografice pe baza unor servicii Web specializate – vor putea fi exportate in formatele CSV, WebP si SVG.
